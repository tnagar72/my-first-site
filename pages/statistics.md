# Project Statistics

This page demonstrates custom Jekyll plugins in action!

## Portfolio Overview

- **Total Projects:** {% project_stats total %}
- **Completed Projects:** {% project_stats completed %}
- **In Progress:** {% project_stats in_progress %}
- **Technologies Used:** {% project_stats technologies %}

## How This Works

The statistics above are automatically generated by a custom Jekyll plugin that:

1. **Analyzes your projects** during site build
2. **Calculates statistics** from project front matter
3. **Provides Liquid tags** for displaying the data
4. **Generates JSON files** with project data

## Plugin Features Demonstrated

### Custom Liquid Filters
- `reading_time` filter for blog posts
- Automatic word count and time estimation

### Custom Liquid Tags
- `project_stats` tag for displaying project statistics
- Dynamic content generation from collections

### Jekyll Hooks
- `post_write` hook for generating additional files
- Automatic JSON data file creation

## Advanced Plugin Concepts

**Plugin Types:**
- **Generators** - Create new pages/posts
- **Converters** - Process file types
- **Tags** - Custom Liquid tags
- **Filters** - Custom Liquid filters
- **Hooks** - Run code at specific build stages

**Real-world Plugin Examples:**
- SEO optimization
